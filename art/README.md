# Artwork

Author: **cocomelonc**

All game scenery, tiles, icons, the hedgehog, particles, and interface shapes
are original procedural drawings rendered at runtime by
`HedgehogGardenView.java`. They require no downloaded tileset and are licensed
with the project under the MIT License.

The only bundled visual dependency is the Nunito typeface. It is available
under the SIL Open Font License 1.1; the exact license text is preserved in
`third_party/nunito/OFL.txt`.
